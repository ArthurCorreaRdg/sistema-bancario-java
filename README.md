# 💰 Sistema Bancário em Java

Projeto desenvolvido com foco em Programação Orientada a Objetos (POO).

## 🚀 Funcionalidades

- Depósito com validação
- Saque com limite e saldo
- Transferência entre contas
- Tipos de conta:
  - Corrente
  - Salário
- Registro de transações (extrato)
- Uso de ENUM para regras de negócio

## 🧠 Conceitos aplicados

- Encapsulamento
- Herança
- Polimorfismo
- Domain-Driven Design (básico)

## 📌 Tecnologias

- Java
- BigDecimal
